# SmartExpensePro

A simple *web-based expense tracker* built with *HTML, CSS, and JavaScript*.  
Track daily expenses, see total spending, and manage your budget.

---

## Features

- Add and delete expenses  
- Category-wise total  
- Real-time calculation  
- Data saved in browser (LocalStorage)  

---

## Tech Stack

HTML | CSS | JavaScript

# How to Use

1. **Add an Expense**:
   - Enter the expense name in the "Expense name" field.
   - Enter the amount in the "Amount" field.
   - Click the "Add Expense" button.

2. **Delete an Expense**:
   - Click the "Delete" button next to the expense you want to remove.

3. **View Total Spending**:
   - The total amount spent is displayed at the bottom of the page.
